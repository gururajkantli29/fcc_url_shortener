# URL Shortener API

## Overview
A URL shortener microservice API project for freeCodeCamp.

## Project Structure
- `index.js` - Main Express server entry point
- `views/index.html` - Frontend HTML page
- `public/style.css` - Stylesheet

## Tech Stack
- Node.js 20
- Express.js
- CORS middleware

## Configuration
- Port: 5000 (default)
- Host: 0.0.0.0

## API Endpoints
- `GET /` - Serves the frontend page
- `GET /api/hello` - Test endpoint returning JSON greeting

## Running Locally
```bash
npm start
```
